# Top100Leetcode

LeetCode热门前100道算法题的JavaScript解法和详细解释。

## 项目结构

```
Top100Leetcode/
├── README.md              # 项目说明文件
└── problems/              # 题目分类目录
    ├── array/             # 数组相关题目
    ├── linkedlist/        # 链表相关题目
    ├── tree/              # 树相关题目
    ├── dynamic-programming/ # 动态规划相关题目
    ├── two-pointers/      # 双指针相关题目
    ├── hash-table/        # 哈希表相关题目
    ├── stack/             # 栈相关题目
    ├── queue/             # 队列相关题目
    ├── backtracking/      # 回溯算法相关题目
    ├── greedy/            # 贪心算法相关题目
    ├── binary-search/     # 二分查找相关题目
    ├── bit-manipulation/  # 位操作相关题目
    ├── graph/             # 图相关题目
    ├── string/            # 字符串相关题目
    ├── divide-and-conquer/ # 分治算法相关题目
    ├── math/              # 数学相关题目
    ├── depth-first-search/ # 深度优先搜索相关题目
    ├── breadth-first-search/ # 广度优先搜索相关题目
    ├── union-find/        # 并查集相关题目
    ├── sliding-window/    # 滑动窗口相关题目
    ├── trie/              # 字典树相关题目
    └── design/            # 设计相关题目
```

## 题目分类

1. **数组**：包含数组相关的算法题，如两数之和、三数之和等。

8. **队列**：包含队列相关的算法题，如滑动窗口最大值、设计循环队列等。
9. **回溯**：包含回溯算法相关的算法题，如全排列、子集等。
10. **贪心**：包含贪心算法相关的算法题，如跳跃游戏、买卖股票的最佳时机等。
11. **二分查找**：包含二分查找相关的算法题，如二分查找、搜索旋转排序数组等。
12. **位操作**：包含位操作相关的算法题，如位1的个数、汉明距离等。
13. **图**：包含图相关的算法题，如岛屿数量、课程表等。
14. **字符串**：包含字符串相关的算法题，如回文串、字符串匹配等。
15. **分治**：包含分治算法相关的算法题，如多数元素、合并K个排序链表等。
16. **数学**：包含数学相关的算法题，如罗马数字转整数、整数反转等。
17. **深度优先搜索**：包含深度优先搜索相关的算法题，如路径总和、二叉树的最大深度等。
18. **广度优先搜索**：包含广度优先搜索相关的算法题，如二叉树的层序遍历、单词接龙等。
19. **并查集**：包含并查集相关的算法题，如省份数量、岛屿数量等。

## 使用方法

1. 克隆项目到本地：
   ```bash
   git clone <项目地址>
   ```

2. 进入项目目录：
   ```bash
   cd Top100Leetcode
   ```

3. 运行题目解法：
   ```bash
   node problems/array/two-sum.js
   ```
