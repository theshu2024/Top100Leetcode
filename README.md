# Top100Leetcode

LeetCode热门前100道算法题的JavaScript解法和详细解释。

## 项目结构

```
Top100Leetcode/
├── README.md              # 项目说明文件
└── problems/              # 题目分类目录
    ├── array/             # 数组相关题目
    ├── linkedlist/        # 链表相关题目
    ├── tree/              # 树相关题目
    ├── dynamic-programming/ # 动态规划相关题目
    ├── two-pointers/      # 双指针相关题目
    ├── hash-table/        # 哈希表相关题目
    ├── stack/             # 栈相关题目
    ├── queue/             # 队列相关题目
    ├── backtracking/      # 回溯算法相关题目
    ├── greedy/            # 贪心算法相关题目
    ├── binary-search/     # 二分查找相关题目
    ├── bit-manipulation/  # 位操作相关题目
    └── graph/             # 图相关题目
```

## 题目分类

1. **数组**：包含数组相关的算法题，如两数之和、三数之和等。
2. **链表**：包含链表相关的算法题，如反转链表、合并链表等。
3. **树**：包含树相关的算法题，如二叉树遍历、二叉搜索树操作等。
4. **动态规划**：包含动态规划相关的算法题，如最长递增子序列、背包问题等。
5. **双指针**：包含双指针相关的算法题，如盛最多水的容器、接雨水等。
6. **哈希表**：包含哈希表相关的算法题，如有效的字母异位词、两数之和等。
7. **栈**：包含栈相关的算法题，如有效的括号、最小栈等。
8. **队列**：包含队列相关的算法题，如滑动窗口最大值、设计循环队列等。
9. **回溯**：包含回溯算法相关的算法题，如全排列、子集等。
10. **贪心**：包含贪心算法相关的算法题，如跳跃游戏、买卖股票的最佳时机等。
11. **二分查找**：包含二分查找相关的算法题，如二分查找、搜索旋转排序数组等。
12. **位操作**：包含位操作相关的算法题，如位1的个数、汉明距离等。
13. **图**：包含图相关的算法题，如岛屿数量、课程表等。

## 使用方法

1. 克隆项目到本地：
   ```bash
   git clone <项目地址>
   ```

2. 进入项目目录：
   ```bash
   cd Top100Leetcode
   ```

3. 运行题目解法：
   ```bash
   node problems/array/two-sum.js
   ```
